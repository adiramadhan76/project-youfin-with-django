# project-youfin-with-django
YouFin (Young Finance) adalah aplikasi berbasis web yang dirancang untuk membantu mahasiswa dan generasi muda mengelola keuangan pribadi secara mudah dan efisien melalui fitur pencatatan pengeluaran, perencanaan anggaran, dan tips keuangan. Dengan antarmuka yang ramah pengguna, YouFin memudahkan pengguna dalam melacak dan mengelola keuangan secara real time serta membangun kebiasaan finansial yang bijak. Aplikasi ini dirancang dan dikembangkan selama program magang MBKM MSIB Batch 7.
